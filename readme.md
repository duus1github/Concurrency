# 全书总结:
## 全书四大经典模型
- 1.阻塞安全队列(线程池底层)
- 2.标准生产者,消费者
- 3.通用可返回值线程池
- 4.可启停+ 绑核常驻工作线程

## 并发编程黄金10条铁率
- 1.尽量少用共享数据,共享即麻烦
- 2.共享数据优先加锁,有限RALL自动管理锁
- 3.wait条件变量必须while循环判断
- 4.线程传参严防引用悬空,临时变量失效
- 5.临界区代码越短越好,禁止休眠,IO,复杂运算
- 6.多锁遵循固定顺序,杜绝死锁
- 7.线程数量贴近CPU核心数,避免泛滥创建
- 8.高频计数优先atomic,不滥用互斥锁
- 9.追求稳定优先,性能其次,不出BUG第一
- 10.业务层少手写无锁,底层框架在使用
