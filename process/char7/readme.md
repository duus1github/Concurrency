# 第 七 章 线程thread 核心学习
## 7.1 线程基础概念
### 1.定义
线程是进程内的执行单元，也叫轻量级进程
一个进程可以包含多个进程，共享同一个地址空间
### 2.原理
- 进程：独立地址空间，独立资源
- 线程：共享进程资源（代码，数据，文件，信号）
- 线程切换比进程快得多
### 3.作用/场景
- 高并发服务（减少资源消耗）
- 后台服务+前台任务并行
- 多任务分工协作


## 7.2 线程创建、退出、等待(最基础API)
### 4.api
pthread_create() 创建线程
pthread_exit()   退出线程
pthread_joint()  等待线程结束
pthread_mutex_t  互斥锁

## 7.3 线程共享资源与互斥锁
### 1.定义
互斥锁（mutex）:保证多线程同时访问共享资源时只有一个线程在操作，防止数据混乱
### 2.原理
- 加锁：进入临界区
- 解锁：退出临界区
- 其他线程阻塞等待
### 3.作用/场景
- 全局变量并发修改
- 共享数据保护
- 线程安全必备

## 7.4 线程同步（条件变量）
### 1.定义
条件变量用来让线程等待某个条件满足，然后被唤醒。
### 2.原理
- pthread_cond_wait():等待+自动释放锁
- pthread_cond_signal():唤醒一个等待的线程
### 3.作用/场景
- 生产者消费者模型
- 任务队列
- 线程间同步
