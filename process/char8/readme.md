# 第八章 线程池（thread pool） 企业级高并发核心
## 8.1 线程池概念
### 1.定义
线程池 = 预先创建好一批固定线程+一个任务队列
线程不断从队列取任务，不用频繁创建/销毁线程
### 2.原理
1、启动时创建n个线程，一直运行(常驻)
2、任务放入队列
3、线程竞争取任务执行
4、任务执行完不退出，继续取下一个
5、销毁时统一释放
### 3.作用/场景
- 高并发服务器（避免频繁创建线程）
- 任务处理、异步任务
- 性能提升巨大，是企业及标准

## 8.2 线程池核心组成
1.任务队列：存放待执行任务
2.工作线程：循环取任务执行
3.互斥锁：保护队列
4.条件变量：队列空时线程等待
